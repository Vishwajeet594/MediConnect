# MediConnect

MediConnect is a healthcare appointment booking platform with React frontend and Node.js/Express backend. It supports doctor discovery, appointment scheduling, user authentication, and doctor profile details.

## Application Flow

- User lands on the homepage and can browse available doctors.
- User signs up or logs in to access protected appointment booking features.
- User searches for or selects a doctor to view detailed profile information.
- User chooses a consultation type, date, and available time slot.
- User confirms the appointment booking through the doctor detail page.
- The backend stores the appointment and the user can manage bookings through the app.
